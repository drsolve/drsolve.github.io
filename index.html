Core Capabilities

What DixonRes Does

DixonRes provides multiple computational modes for polynomial system analysis over finite fields, from basic resultant computation to full system solving with ideal reduction.

∆
Dixon Resultant

Compute the Dixon resultant for variable elimination in polynomial systems. Supports multivariate systems with any number of elimination variables.

Core Mode
∑
System Solver

Solve well-determined n×n polynomial systems (n equations in n unknowns) over prime or extension finite fields. Full solution enumeration.

--solve
⊕
Ideal Reduction

Dixon resultant computation with triangular ideal reduction. Specify ideal generators to reduce the system before elimination.

Advanced
𝔽
Prime Fields F_p

Full support for prime finite fields F_p where p < 2⁶³. Uses FLINT's high-performance modular arithmetic throughout all computations.

Fast Path
𝔾
Extension Fields F_{p^k}

Arithmetic in extension fields such as GF(2⁸), GF(3⁵), etc. Supports custom irreducible polynomials or FLINT's built-in defaults.

Extension
⚡
PML Acceleration

Optional PML library integration for polynomial matrix operations. Automatically activated when available, accelerating well-determined prime field systems.

Optional

Usage Examples

DixonRes provides a unified CLI interface with multiple operational modes. All modes automatically write results to a timestamped output file.

bash — Dixon Resultant (Basic) 
# Syntax: ./dixon "polynomials" "eliminate_vars" field_size
# Number of eliminate_vars = number of equations - 1

./dixon "x+y+z, x*y+y*z+z*x, x*y*z+1" "x,y" 257

# Output: resultant polynomial in remaining variable z
# Written to: solution_YYYYMMDD_HHMMSS.dat
bash — Polynomial System Solver 
# Syntax: ./dixon --solve "polynomials" field_size
# n equations in n unknowns

./dixon --solve "x^2 + y^2 + z^2 - 6, \
              x + y + z - 4, \
              x*y*z - x - 1" 257

# Variables are auto-detected from polynomial expressions
# All solutions over GF(257) are enumerated

# Silent mode (no console output, file still written):
./dixon --silent --solve "..." 257
bash — Extension Fields 
# Default extension field (FLINT built-in polynomial)
# t is the extension field generator by default
./dixon "x + y^2 + t, x*y + t*y + 1" "x" 2^8

# Custom irreducible polynomial (e.g., AES field F₂₅₆)
./dixon --solve "x^2 + t*y, x*y + t^2" \
        "2^8: t^8 + t^4 + t^3 + t + 1"

# Other extension examples
# GF(3^5):  "3^5"  or  "3^5: t^5 + 2*t + 1"
# GF(2^16): "2^16: t^16 + t^5 + t^3 + t^2 + 1"
bash — Ideal Reduction Mode 
# Syntax: ./dixon "polys" "elim_vars" "ideal_gens" "all_vars" field

./dixon \
  "a1^2 + a2^2 + a3^2 + a4^2 - 10, \
   a4^3 - a1 - a2*a3 - 5" \
  "a4" \
  "a2^3 = 2*a1 + 1, \
   a3^3 = a1*a2 + 3, \
   a4^3 = a1 + a2*a3 + 5" \
  257

# Ideal generators reduce the system before Dixon computation
# Useful for structured systems (e.g., from block ciphers)
text — File Input Format 
# Dixon Mode file (example.dat):
257                        # Line 1: field size
x+y+z                      # Line 2+: polynomials
x*y+y*z+z*x
x*y*z+1
x,y                        # Last: variables to eliminate

# Solver Mode file (system.dat):
257                        # Line 1: field size
x^2 + y^2 + z^2 - 6       # Line 2+: n equations
x + y + z - 4
x*y*z - x - 1

# Run with:
./dixon example.dat
./dixon --solve system.dat

# Output: example_solution_YYYYMMDD_HHMMSS.dat
Arithmetic Domains

Finite Field Support

DixonRes supports both prime fields and extension fields. Prime fields with optional PML acceleration offer the best performance.

𝔽ₚ

Prime Fields

Finite fields GF(p) for any prime p < 2⁶³. All basic arithmetic is performed modulo p using FLINT's optimized routines. When the PML library is available, polynomial matrix operations are automatically accelerated for well-determined systems.

./dixon --solve "..." 257
./dixon --solve "..." 65537
./dixon --solve "..." 4294967311
𝔽ₚᵏ

Extension Fields

Finite fields GF(pᵏ) specified as p^k. The generator t satisfies a chosen irreducible polynomial over GF(p). FLINT's built-in polynomial is used by default; custom polynomials can be specified after a colon.

./dixon "..." 2^8
./dixon "..." "2^8: t^8+t^4+t^3+t+1"
./dixon "..." 3^5

Getting Started

Installation

DixonRes requires FLINT 3.4.0 or later. The optional PML library can be installed alongside for performance improvements on prime field systems.

ℹ FLINT 3.4.0+ is required. PML is optional but recommended for well-determined systems over prime fields.
01

Install FLINT

Get FLINT 3.4.0 or newer from the official repository. Follow the build instructions for your platform.

  • FLINT ≥ 3.4.0 · required
  • PML latest · optional
02

Clone the Repository

Get the source from GitHub.

git clone https://github.com/DixonRes/DixonRes
cd DixonRes
03

Configure & Build

Run the configure script to detect dependencies, then build with make.

./configure
make
make install # optional
04

Verify Installation

Test with the quick example to confirm everything is working correctly.

./dixon --solve \
  "x+y-3, x*y-2" 257
